Hem CGMiner på Ubuntu Statistik

Installera CGMiner på Ubuntu

(Ubuntu/Xubuntu/Kubuntu/Ubuntu MATE, med mera)

(AMD GPU)

i386/AMD64

1. Förberedelse

sudo apt-get install libcurl4-openssl-dev pkg-config libtool libncurses5-dev libudev-dev screen xterm
sudo apt-get install git-core
sudo apt-get install openssh-server byobu
sudo apt-get install autoconf
sudo apt-get install automake

2. Hämta CGMiner

git clone https://github.com/ckolivas/cgminer/tree/3.7

3. Installera AMD drivrutiner

cd /opt
sudo tar xvzf /home/användarnamn/AMD-APP-SDK-v2.8.1.0-lnx32.tgz
cd /
sudo tar xvzf /opt/icd-registration.tgz
cd /opt
sudo tar xvzf AMD-APP-SDK-v2.8.1.0-RC-lnx32.tgz
sudo ln -s /opt/AMD-APP-SDK-v2.8.1.0-RC-lnx32/include/CL
/usr/include
sudo ln -s /opt/AMD-APP-SDK-v2.8.1.0-RC-lnx32/lib/x86/* /usr/lib/

4. Installera ADL drivrutiner

mkdir ADL
mv ADL_SDK_5.0.zip ADL
cd ADL
unzip ADL_SDK_5.0.zip
sudo cp include/*.h /home/användarnamn/cgminer/ADL_SDK

5a. Modifiera CGMiner med Bitcoin och Litecoin (scrypt) stöd

cd cgminer
./autogen.sh #Behövs bara om du utgår från git
CFLAGS="-O2 -Wall -march=native -I/opt/AMD-APP-SDK-v2.8.1.0-RC-lnx32/include" LDFLAGS="-L/opt/AMD-APP-SDK-v2.8.1.0-
RC-lnx32/lib/x86" ./configure --enable-opencl --enable-scrypt

5b. Fortsättnig på output

sudo make
sudo make install

6. Installera AMD Catalyst drivrutiner (fglrx)

unzip amd-catalyst-13.8-beta2-linux-x86.x86_64.zip
chmod 700amd-catalyst-13.8-beta2-linux-x86.x86_64.run
sudo ./amd-catalyst-13.8-beta2-linux-x86.x86_64.run

Starta om din dator efter steg 6!

7. Konfiguera drivrutiner

sudo aticonfig --adapter=all --initial
sudo aticonfig --lsa
sudo aticonfig --adapter=all --odgt

8. Starta CGMiner!

./cgminer
